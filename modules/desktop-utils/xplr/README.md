# xplr File Manager Configuration

Configuration for [xplr](https://github.com/sayanarijit/xplr), a hackable, fast, and interactive terminal file explorer.

## Contents

- `init.lua` - Full xplr configuration with custom modes, keybindings, and UI customization

## Usage

This module is part of the desktop utilities bundle. The config is synced to `~/.config/xplr/init.lua` via nix-repo-sync. See the [desktop-utils README](../README.md) and [Desktop Environment Management](../../desktops/README.md) for how desktop modules are layered.

## Features

- Custom file operations (create, rename, delete, copy, move)
- Color-coded file type indicators
- Custom input modes for power-user workflows
- Symlink and permission handling
